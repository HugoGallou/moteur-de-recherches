    Le référencement web

    Le référencement passe nécessairement par une phase d’indexation, durant laquelle une base de données est constituée. Le référencement défini donc aussi l’action d’inscrire une chose dans une base de donnée pour que cette dernière puisse ensuite être retrouvée / consultée.
